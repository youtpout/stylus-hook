Stylus Hook

Welcome to stylus hook, the aim of this project is to create hooks for uniswap v4 directly with Stylus on Arbitrum.

Why use stylus to create hooks ?

The introduction of hooks has opened up an infinite range of possibilities for defi, but as hooks become more complex and handle more and more data, their gas cost becomes exponential, to the point where it will be very costly, if not impossible, to use certain pools because of the hook attached.
Thanks to stylus, we can push back the limits of what's possible with hooks, making hooks much more complex and managing much more data without necessarily being expensive in terms of gas.

Actual State

This project was created as part of a hackathon. Unfortunately, given the limited time available for ahackathon, the project hasn't completely achieved its main objective, which was to create hooks with stylus.
It was also necessary to use a different chain from arbitrum sepolia to deploy smartcontracts in stylus, which was not compatible with the latest version of Solidity required for uniswap v4 to function properly, so a fork of uniswap v4 that differed from the official version had to be created.

Now that stylus is directly usable on the arbitrum sepolia testnet, which supports the latest version of solidity, it will be directly integrated with the official version of uniswap v4.

Uniswap v4 status

As Uniswap v4 is not yet finalized, the project will have to be continually updated until the final version of uniswap v4 is delivered. That's why the audit can't be done before the end of the year, to make sure that the basehooks created will be compatible with the latest version of uniswap v4.

Uniswap v4 in numbers

Uniswap has achieved 2 trillion in volume since its creation.
Uniswap supports more than 11 networks.
Arbitrum surpasses $150 billion in total transaction volume on Uniswap.

The goal

The aim is for end users to be able to create hooks on Uniswap v4 using stylus, and as the contract will be audited, this will give certain users greater peace of mind from a security point of view.
As stylus will be much more competitive from a performance and cost point of view, some large projects may be carried out only on Arbitrum, increasing the attraction and use of this blockchain.
